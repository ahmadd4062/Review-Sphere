# Review Sphere

Review Sphere is a full-stack Django web application for creating, managing, and analyzing user reviews. It includes authentication, blog/review management, dashboard features, and sentiment analysis.

## Features
- User authentication (Signup/Login/Logout)
- Create, update, delete reviews/blogs
- User dashboard
- Contact system
- Sentiment analysis module

## Tech Stack
- Django (Python)
- HTML, CSS, Bootstrap
- SQLite
- JavaScript

## Setup Instructions
1. Clone repository  
2. Install requirements:
   pip install -r requirements.txt
3. Run migrations:
   python manage.py migrate
4. Start server:
   python manage.py runserver
